# SpendWise 💰
A full stack expense tracker web application.

## Live Demo
🔗 [spendwise-by-huzaifa.up.railway.app]

## Features
- Add and delete expenses
- categorize by Food, Rent, Transport, Health, Other
- Visual donut chart showing spending breakdown
- Real time summary — total, largest expense, daily average
- Data saved to SQLite database permanently

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript, Chart.js
- **Backend:** Python, Flask
- **Database:** SQLite
- **Deployed on:** Railway.app

## How to Run Locally
```bash
pip install flask flask-cors
python app.py
```
Then open http://127.0.0.1:5000
